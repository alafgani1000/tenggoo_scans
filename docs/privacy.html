Kebijakan Privasi

Terakhir diperbarui: 29 April 2026

🔒 TenggooScans tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda ke server manapun. Semua proses berjalan sepenuhnya di perangkat Anda.

1. Tentang Aplikasi

TenggooScans adalah aplikasi pemindai dokumen Android yang dikembangkan secara independen. Aplikasi ini memungkinkan pengguna untuk memindai dokumen, mengonversi gambar ke PDF, dan mengekstrak halaman PDF langsung dari perangkat.

2. Data yang Dikumpulkan

Kami tidak mengumpulkan data pribadi apapun. Aplikasi ini berjalan sepenuhnya secara offline dan tidak mengirimkan data ke server eksternal.

  • Tidak ada akun pengguna yang diperlukan
  • Tidak ada data yang dikirim ke internet
  • Tidak ada analitik atau pelacakan perilaku pengguna
  • Tidak ada iklan pihak ketiga

3. Izin Aplikasi

Aplikasi memerlukan beberapa izin perangkat untuk berfungsi dengan baik:

  • Kamera: Digunakan untuk memindai dokumen melalui Google ML Kit Document Scanner. Data dari kamera hanya diproses secara lokal di perangkat Anda.
  • Penyimpanan: Digunakan untuk menyimpan hasil pemindaian (PDF/gambar) ke folder Downloads perangkat Anda.
  • Galeri (READ_MEDIA_IMAGES): Digunakan untuk memilih gambar dari galeri sebagai input konversi.

4. Google ML Kit Document Scanner

Fitur pemindaian menggunakan Google ML Kit yang beroperasi di perangkat (on-device). Pemrosesan gambar dilakukan secara lokal dan tidak dikirimkan ke server Google. Silakan merujuk pada Kebijakan Privasi Google untuk informasi lebih lanjut mengenai ML Kit.

5. Data Pihak Ketiga

Aplikasi tidak mengintegrasikan layanan pihak ketiga yang mengumpulkan data pengguna (seperti Firebase, Crashlytics, AdMob, atau sejenisnya).

6. Keamanan Data

Karena semua data diproses secara lokal di perangkat Anda, keamanan file hasil pemindaian sepenuhnya bergantung pada keamanan perangkat Anda sendiri. Kami menyarankan untuk mengaktifkan kunci layar pada perangkat Anda.

7. Perubahan Kebijakan

Jika terjadi perubahan pada kebijakan ini, halaman ini akan diperbarui. Versi terbaru selalu tersedia di scan.tenggoo.my.id/privacy.html.

8. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai kebijakan privasi ini, silakan buka halaman Issues di repositori GitHub kami.
